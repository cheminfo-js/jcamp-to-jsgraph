# jcamp-to-jsgraph

  [![build status][travis-image]][travis-url]

Converts a jcamp to a jsgraph json format.

## Installation

`$ npm install --save jcamp-to-jsgraph`

## Usage

```js
import library from 'jcamp-to-jsgraph';

const result = library(args);
// result is ...
```
